# Front desk web UI (SvelteKit)

This is the main operator UI for the Front Desk App. It is built with SvelteKit and, in development, proxies `/api` to the Express server (`npm run server` from the repo root, port 3000).

- **Run (from repo root):** `npm run dev`
- **Build (from repo root):** `npm run build` (outputs to `apps/web/build`)
- **Project docs:** see the root [README.md](../README.md), [QUICK_START.md](../QUICK_START.md), and [STAFF_GUIDE.md](../STAFF_GUIDE.md).

The default template text from `create-svelte` / `sv` is intentionally replaced by this file.
